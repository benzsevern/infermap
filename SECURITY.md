# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in infermap, please report it responsibly:

1. **Do not** open a public GitHub issue
2. Use [GitHub Security Advisories](https://github.com/benzsevern/infermap/security/advisories/new) to report privately
3. Include steps to reproduce and potential impact

You should receive a response within 48 hours.
